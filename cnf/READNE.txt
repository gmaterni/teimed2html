bin/
----------------------------------
esecuzione  le fasi del projetto
(per semplicitÃ  tutti gli scritti fanno riferimento
al manoscritto par)
----------------------------------
bin/splitxml.json

estraee dal file:
xml/florpar.xml

gli episodii:
xml/par/ep1.xml
xml/par/ep2.xml
...
produce lista degli episdofi da includre successivamente
xml/par/par_list.xml
----------------------------------------
bin/dirxml2html_di.json

per ogni dir dei manoscritti (par,tor,tou,ven, ..)
trasforma da xml a html i singoli episodi
contenebti sia l'interpretativa che la diplonatica

xml/par/ep1.xml => html/par/txt/ep1.html
xml/par/ep2.xml => html/par/txt/ep2.html
......

per effettuare controlli 
(da NON utilizzatre in produzione)
peoduce per ogni episodio
html formattato per la

diplomatica
html/par/txt/ep1d_F.html

interpretativa
html/par/txt/ep1i_F.html
---------------------------------------
bin/pannel_list.json

produce per ogni manoscritto (par,tor,..)
la lista degli episodi per la

diplomatica
xml/par/par_list.xml => html/par/txt/par_listd.html

interpretativa
xml/par/par_list.xml => html/par/txt/par_listi.html
---------------------------------------
bin/pannel_par.json

nel file:
html/txt_pannel.html

sosiusce i parametri con i file
EPISODE_LIST_DIPL => html/par/txt/par_listd.html

EPISODE_LIST_INTER => html/par/txt/par_listi.html

scrive il  file contenente le intestazioni e l'elenco 
html/par/txt/par.html
